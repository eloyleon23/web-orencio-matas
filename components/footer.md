# Footer Component

Objetivo:
Crear un footer corporativo premium para una empresa industrial especializada en droguería, perfumería, pinturas y suministros para carrocería.

El footer debe transmitir:
- profesionalidad
- estabilidad
- empresa consolidada
- distribución industrial moderna

---

# Estructura del Footer

## Layout

Footer dividido en 4 columnas:

1. Branding
2. Navegación
3. Servicios
4. Contacto

---

# Estilo Visual

Características:
- fondo negro grafito
- tipografía clara
- detalles en rojo corporativo
- iconografía moderna
- mucho espacio visual
- diseño limpio y elegante

Inspiración:
- automoción premium
- industria moderna
