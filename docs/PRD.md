# PRD — Proyecto Web Corporativa
## Orencio Matas y Hnos, S.L.

---

# 1. RESUMEN DEL PROYECTO

## Objetivo

Desarrollar una demo funcional, moderna y profesional de la nueva página web corporativa de Orencio Matas y Hnos, S.L., utilizando WordPress + Elementor Pro bajo metodología BMAD y desarrollo asistido con Windsurf.

El objetivo inicial es construir un MVP visual y funcional que permita:

- Presentar una imagen moderna al cliente
- Validar diseño y arquitectura
- Construir una base escalable
- Definir branding digital
- Preparar futuras evoluciones
